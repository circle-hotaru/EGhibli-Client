# EGhibli
EGhibli是一个简单的在线展示Ghibli相关资讯平台Demo，项目采用Spring Boot + Vue开发

## 技术栈
### 前端技术栈
- Vue
- Bootstrap
- vue-router
- vue-lazyload

### 后端技术栈
- Maven
- SpringBoot
- MySQL
